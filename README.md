# Laptop Sales Analysis Dashboard

This Power BI dashboard provides a comprehensive analysis of laptop sales performance by transforming raw sales data into meaningful business insights through interactive visualizations. It highlights key metrics such as total sales revenue (3.44M), product count, average pricing, and customer ratings to evaluate overall business growth.

The dashboard includes brand-wise filtering for detailed analysis, enabling users to compare top laptop brands based on sales and performance. Visual elements such as rating trends, product specification distribution (color, graphics, CPU, OS, hard disk, and model), and stock vs sales performance help identify customer preferences, product demand, and inventory efficiency.

Designed with a professional dark-themed interface, this project demonstrates strong data visualization, dashboard design, and business intelligence skills using Power BI. It supports data-driven decision-making by uncovering trends, optimizing product strategies, and improving sales performance.
